# Design + Technology Presentations

> **Speaker Portfolio** | Trying to bridge the gap between design and technology through engaging presentations and workshops

## About

This repository contains a curated collection of presentations and workshops I have given focused on the intersection of design and technology. Each presentation is to educate, inspire, or demonstrate practical applications of modern (design) tools and (development) practices.

## 🎯 Speaking Topics

### 🎨 [Figma Design Systems](./figma/)

- **Figma Fundamentals** - Introduction to Figma's collaborative design platform and why it's become the industry standard
- **Component Systems** - Building reusable components with variants, auto layout, and component properties
- **Design System Thinking** - Understanding the difference between component libraries and design systems
- **Hands-on Component Creation** - Practical exercises building button components with multiple variants and states
- **Design System Resources** - Curated collection of tools, guides, and experts in the design systems space

### 🌐 **[Accessibility Workshop](./accessibility/)**

- **Accessibility Workshop** - Hands-on sessions covering WCAG guidelines, testing methodologies, and inclusive design principles
- **WCAG Fundamentals** - Understanding the POUR principles (Perceivable, Operable, Understandable, Robust) and compliance levels
- **European Accessibility Act** - Legal requirements and upcoming deadlines (June 2025)
- **Real-world Accessibility Issues** - Common problems like contrast ratios, missing alt text, and empty links
- **Practical Implementation Tips** - Making interactive elements descriptive, using skip links, and respecting user preferences

### 🔌 [MQTT & IoT](./mqtt/)

- **MQTT Fundamentals** - Understanding publish/subscribe messaging patterns and lightweight protocols
- **Topic Structure & Wildcards** - Learning hierarchical topic naming and subscription patterns
- **Live Interactive Examples** - Real-time chat, collaborative tools, and democratic interfaces
- **Arduino Integration** - Connecting physical devices to web applications via MQTT
- **Browser Event Publishing** - Capturing mouse, keyboard, and scroll events for real-time communication

### 🤖 [AI-Assisted Development](./vibing-with-ai/)

- **Vibe Coding Concept** - Understanding Andrej Karpathy's vision of AI-assisted development workflows
- **AI Workflow Spectrum** - From prompt engineering to agentic AI, understanding different collaboration models
- **Context-Driven Development** - How providing better context leads to better AI assistance
- **Real-world AI Integration** - Practical demonstrations of AI tools in design and development workflows
- **The Future of Programming** - Exploring how AI is changing the nature of software development

## 📁 Presentation Structure

Each presentation is built with [Slidev](https://sli.dev/) and may include:

- **Interactive slides** with live code examples
- **Hands-on workshops** with practical exercises
- **Export-ready formats** (PDF, HTML) for sharing
- **Component libraries** demonstrating concepts
- **Code snippets** and implementation examples

### 🚀 Quick Start

Each presentation can be run independently:

```bash
# Navigate to any presentation folder
cd accessibility  # or figma, mqtt, vibing-with-ai

# Install dependencies
npm install

# Start the presentation
npm run dev

# Export to PDF
npm run export
```
